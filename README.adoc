== Deploy maven site

=== Pre-requisite

Open your Maven settings (~/.m2/settings.xml) and add the following server configuration:

[source, xml]
----
<server>
    <id>github</id>
    <username>YOUR_GITHUB_USERNAME</username>
    <password>YOUR_PASSWORD</password>
</server>
----

=== Command

[source]
----
mvn clean post-site -Pdeploy-site
----
